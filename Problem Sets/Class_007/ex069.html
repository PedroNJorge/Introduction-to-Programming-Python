[IP069] Items Removal

Regular-Sized Rudy, a frequent visitor to Bob's Burgers, has a peculiar dislike for menu items that start with the letter "B" or "S". He wants to filter these out from a set of available menu items. The menu items are given as a set of strings, where each string represents the name of a dish. He needs your help to remove from the menu any items starting with a certain letter.

The Problem

Write a function remove(x, menu) that given a lowercase letter x and a set of lowercase strings representing menu items, removes from the set all items that start with the letter x.

Constraints

The following limits are guaranteed in all the test cases that will be given to your program:

1 ≤ |menu| ≤ 100       number of items in the menu

Example Function Calls Example Output 
menu = {"banana", "juice", "burger", "coffee", "tea"}
remove("b", menu)
print(type(menu))
print(sorted(list(menu)))
