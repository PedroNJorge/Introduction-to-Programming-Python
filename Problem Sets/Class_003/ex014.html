[IP014] More Circle Math

Understanding the properties of circles is essential in geometry. In this exercise, you will help users calculate various attributes of a circle based on its radius.

The Problem

Write a program that allows users to choose one of three options to calculate a property of a circle based on the given radius r. Use math.pi in your calculations.

Option Calculation
1 Calculate the diameter of the circle.
2 Calculate the perimeter of the circle.
3 Calculate the area of the circle.

If the user enters an invalid option, the program should print "Invalid Option."

Input

The input contains two lines:

  • The first line contains a string representing the user's choice of calculation.
  • The second line contains a floating-point number representing the radius.

Output

The output should display the calculated property of the circle rounded to two decimal places, according to the specified option. Either one of the following:

  • For diameter: "Diameter: [value]"
  • For perimeter: "Perimeter: [value]"
  • For area: "Area: [value]"
  • For an invalid option: "Invalid Option."

Constraints
