[IP057] A Summing Project

Marge Simpson, ever the supportive sister, has been watching Patty and Selma dive into the world of data manipulation with their matrix projects. She’s impressed by how they turned the chaotic hart into a simplified list and then back into a matrix, appreciating the complexity behind what seemed like simple tasks. Inspired by their enthusiasm and recognizing the potential for fun, Marge decides it’s time to try her hand at something a little more challenging herself: summing two matrices!

The Problem

Write a function sum(a, b) that given two matrices a and b (given as lists of lists), both with r rows and c columns, returns a new matrix c which is the result of summing a and b. This matrix should also have r rows and c columns and ci,j should be equal to summing ai,j with bi,j.

Constraints

The following limits are guaranteed in all the test cases that will be given to your program:

1 ≤ r, c ≤ 100       Number of rows and columns
